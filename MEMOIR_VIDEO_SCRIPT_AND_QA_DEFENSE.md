## PART 2: Q&A DEFENSE (30 Anticipated Questions)

### CATEGORY 1: TECHNOLOGY & FEASIBILITY

#### Q1: "Can AI really create a high-quality fragrance? Isn't perfumery an art that requires years of training?"

**ANSWER:**
"Excellent question. The key is that AI doesn't *replace* perfumers—it *assists* them. Here's how it works:

1. **AI handles systematic translation:** Visual elements → olfactive notes (data-driven, repeatable)
2. **Perfumer handles artistry:** Formula balance, note harmony, olfactive excellence (expertise, intuition)

Think of it like architecture: AI can analyze site requirements and suggest structural elements, but the architect designs the building. Both are essential.

**Our beta results:** 82% of AI-suggested formulas were approved or lightly edited by perfumers. By Year 3, with machine learning, we expect 90%+ approval rate.

L'Oréal already uses technology to assist perfumers—Scent-Sation uses neuroscience, Perso uses AI for skincare. MEMOIR extends this to fragrance creation."

---

#### Q2: "How accurate is your visual-to-olfactive mapping? What if the AI misinterprets the image?"

**ANSWER:**
"Our database was built through 100+ hours of interviews with 10 master perfumers. Each mapping is validated by multiple experts:

- If 9/10 perfumers associate 'ocean' with 'salt accord,' relevance score = 0.90
- If only 5/10 agree, score = 0.50 (below our 0.70 threshold, excluded)

**Built-in safety net:** Human perfumer reviews EVERY formula before production. If AI misinterprets, perfumer corrects it.

**User context helps:** We ask users to describe what they remember smelling. 'I remember plumeria flowers' boosts that note's score.

**Continuous improvement:** Our ML model learns from perfumer edits. If perfumers consistently adjust a mapping, the database updates automatically.

**Current accuracy:** 82%. Industry-leading, and it improves with every fragrance we create."

---

#### Q3: "What if a user uploads a photo that's impossible to translate to scent—like an abstract photo or a selfie?"

**ANSWER:**
"Great edge case question. We handle this through:

**1. Image Quality Filter:**
- AI detects if image has insufficient visual elements
- Prompts user: 'This image doesn't have enough environmental context. Try a photo of the location or setting.'

**2. Guided Upload:**
- UI encourages landscape/scene photos vs. close-up selfies
- Examples shown: 'Beach vacation,' 'Garden scene,' 'Café interior'

**3. Context Form:**
- Even if image is limited, user describes memory in text
- 'Where was this?' 'What do you remember smelling?'
- Perfumer can create from description alone if needed

**4. Consultation Option:**
- Premium add-on (+€50): 15-min video call with perfumer
- User walks through memory verbally, perfumer interprets

**In beta testing:** 95% of user-uploaded photos had sufficient context. The 5% that didn't were caught by quality filter."

---

#### Q4: "How do you prevent the AI from creating fragrances that smell bad or cause allergic reactions?"

**ANSWER:**
"Multi-layer safety system:

**1. Approved Ingredient Library:**
- AI can only select from L'Oréal's pre-approved, IFRA-compliant ingredients
- All accords tested for skin safety, allergen levels
- No raw materials that exceed EU/US regulatory limits

**2. Compatibility Rules:**
- Algorithm has built-in 'clash detection' (e.g., won't pair lavender with indole—smells like bathroom cleaner)
- Based on perfumery principles (top/heart/base note structure)

**3. Perfumer Quality Control:**
- Human reviews formula before production
- Perfumer can reject entirely if AI combination doesn't work

**4. Sample-First Policy:**
- Every customer receives 2ml sample vial before opening 30ml bottle
- If unsatisfied, free reformulation within 30 days

**5. Allergen Labeling:**
- All bottles list IFRA allergens (26 required EU declarations)
- Users can flag allergies in profile → AI avoids those ingredients

**Beta testing:** Zero allergic reactions, 90+ NPS on scent quality."
